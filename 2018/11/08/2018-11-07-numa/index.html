Nov 8, 2018

cpu名词详解


socket

主板上面处理器的插槽，每个socket表示一个物理cpu

cpu chip

物理cpu， 看得见摸得着的东西

cpu core

平常所说的多核cpu就是说的cpu core, cpu core有自己单独的l1/l2/l3 cache

logic cpu

就是平常所说的超线程， 单个cpu core里面有两组寄存器/算术逻辑单元，共享浮点运算单元等, 共享部分不冲突的时候，可以并行跑

numa

大学教程上面写的都是访问内存通过北桥芯片的总线模型，但是随着cpu核数越来越多，导致对内存的成为瓶颈，这时候就出现numa架构, 每个cpu都自带内存控制器，将内存划分为多个节点，这样每个cpu 都有自己的本地内存

numa 与 pci 设备

当前cpu没有通过南桥/北桥访问pci设备，可以直连，pci肯定绑定了单个cpu，dma地址也映射到了这个node的内存，所以pci设备也存在本地设备和远程设备的区别

tlb

众所周知，linux下，每个程序都有自己的虚拟地址。当需要访问读写内存时，需要把虚拟地址转换为物理地址，每个程序都有一个页表，里面存储了虚拟内存到物理内存的映射关系，转换的流程是通过mmu完成的，
因为页表由物理内存存储，所以，程序访问一次内存，其实要访问两次物理内存，一次是虚拟到物理地址的转换，另外一次获取真正的内容，为了加快速度，mmu里面开辟了一块告诉缓存，用于存储虚拟内存到物理内存的映射，
但是tlb的大小有限，所以有了dpdk的巨页。
